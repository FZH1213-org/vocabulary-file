## 🎯 功能特性

### 1. 点击发音
- 用户点击紫色背景的词汇标注（如 `spectacular(壮观的)📢`）
- 自动提取英文单词并播放发音
- 使用Web Speech API实现

### 2. 用户体验优化
- **鼠标悬停**：显示手型光标，提示可点击
- **透明度变化**：悬停时透明度降低，提供视觉反馈
- **语速调节**：语速设为0.85，稍慢便于学习
- **音调标准**：音调设为1.0，标准美式发音

## 💻 技术实现

### JavaScript核心代码

```javascript
function speak(word) {
  window.speechSynthesis.cancel();
  var utterance = new SpeechSynthesisUtterance(word);
  utterance.lang = 'en-US';
  utterance.rate = 0.85;
  utterance.pitch = 1.0;
  window.speechSynthesis.speak(utterance);
}
```
