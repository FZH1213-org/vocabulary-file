## 🎯 使用建议

### 1. 按顺序学习
表格按照词汇在故事中首次出现的顺序排列，建议：
- 配合故事文件学习
- 按照故事编号（01-100）顺序学习
- 结合例句理解词汇用法

### 2. 关注高频词汇
出现次数较多的词汇是高频词汇，建议优先掌握：
- always (157次) - 总是
- nervous (132次) - 紧张的
- active (107次) - 积极
- respect (96次) - 尊重
- patient (95次) - 耐心

### 3. 词性学习建议
- 名词（309个）- 记忆事物、概念
- 动词（256个）- 掌握动作表达
- 形容词（158个）- 丰富描述能力
- 副词（36个）- 精准修饰动作

## 💡 学习方法

### 方法一：故事联想法
1. 阅读故事文件（result目录下的学习版HTML）
2. 遇到词汇标记时，查看表格中的详细信息
3. 记忆词汇的音标、词性和例句
4. 复习时使用复习版HTML巩固记忆

### 方法二：高频词汇法
1. 打开Excel表格
2. 按"出现次数"排序
3. 从高频词汇开始学习
4. 结合例句理解用法

### 方法三：词性分类法
1. 按词性筛选词汇
2. 分类学习不同词性的单词
3. 系统化构建词汇体系

## 📖 表格字段说明
