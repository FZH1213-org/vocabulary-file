# 词汇汇总统计报告

## 任务完成情况

**任务目标：** 从100个学习版HTML文件中提取所有词汇，生成Excel表格

**完成时间：** 2026-07-14

---

## 数据统计

### 总体数据
- **处理文件数：** 100个学习版HTML文件
- **提取词汇总数：** 2,778个不同词汇
- **数据完整性：**
  - 有音标的词汇：1,020个（37%）
  - 有词性的词汇：1,142个（41%）
  - 有例句的词汇：2,778个（100%）

### 输出文件

| 文件名 | 格式 | 大小 | 说明 |
|--------|------|------|------|
| 词汇汇总表_扩充版.json | JSON | 1.4MB | 结构化数据，便于程序处理 |
| 词汇汇总表_扩充版.csv | CSV | 908KB | 逗号分隔，Excel可直接打开 |
| 词汇汇总表_扩充版.xls | Excel | 1.1MB | Excel格式，方便查看和编辑 |

### 表格列信息

1. **序号** - 词汇编号（按首次出现顺序）
2. **单词** - 英文单词
3. **音标** - 国际音标（37%的词汇有音标）
4. **词性（中文）** - 词性标注（41%的词汇有词性）
5. **中文释义** - 中文翻译
6. **出现次数** - 在100个故事中出现的总次数
7. **首次出现故事** - 词汇首次出现的章节编号（1-100）
8. **例句** - 包含该词汇的例句

---

## 数据特点

### 词汇分布
- 所有词汇按首次出现故事编号排序
- 部分词汇在多个故事中重复出现
- 词汇覆盖了考研、商务、日常等多个场景

### 音标和词性来源
- 使用免费的Free Dictionary API获取
- 缓存文件：scripts/vocab_cache.json
- 包含1,142个词汇的音标和词性数据

---

## 技术实现

### 处理流程
1. 遍历100个学习版HTML文件
2. 使用正则表达式提取词汇标注
3. 统计词汇出现次数
4. 记录首次出现的故事编号
5. 提取例句
6. 调用API获取音标和词性
7. 生成JSON、CSV、Excel格式输出

### 使用技术
- Node.js 18.19.0
- 正则表达式解析HTML
- Free Dictionary API
- 文件I/O操作
